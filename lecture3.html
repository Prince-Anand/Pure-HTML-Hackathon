Nested Lists in HTML

A nested list is a list that contains another list within it. In HTML, we can create nested lists using the <ul> (unordered list) or <ol> (ordered list) tags.

Each item in the list is represented by the <li> (list item) tag. To create a nested list, we simply place another <ul> or <ol> inside an <li> element.

Example of Nested Lists:

Code1 Output1 Code2 Output2 
<ul>
    <li>Fruits
        <ul>
            <li>Apple</li>
            <li>Mango</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrot</li>
            <li>Broccoli</li>
        </ul>
    </li>
    <li>Dairy
        <ul>
            <li>Milk</li>
            <li>Paneer</li>
        </ul>
    </li>
</ul>

HTML Tables

Tables in HTML are used to display data in a tabular format, consisting of rows and columns. The main tags used to create tables are:

  • <table>: This tag defines the table itself.
  • <tr>: This tag defines a table row.
  • <td>: This tag defines a table cell (data cell) within a row.
  • <th>: This tag defines a table header cell, which is typically bold and centered.
  • <thead>: This tag groups the header content in a table.
  • <tbody>: This tag groups the body content in a table.
  • <tfoot>: This tag groups the footer content in a table.

Example of a Simple Table:

Name Role
Rohit Frontend Developer
Mohit Backend Developer
Anjali Full Stack Developer

In this example, we have a table with two columns: "Name" and "Role". The first row contains the headers, and the subsequent rows contain the data for each individual.

Tables can be further customized with attributes like:

  • border: To add border to the cell and table.
  • cellpadding: Specifies the space between the cell content and its border.
  • cellspacing: To add space between individual table cells.

Tables are useful for organizing and presenting data in a clear and structured manner on web pages.

There are two main attributes used in the <td> & <th>tag to enhance the appearance of tables:
  • colspan: Ek cell ko multiple columns mein spread karne ke liye. (e.g. <td colspan="3">)
  • rowspan: Ek cell ko multiple rows mein spread karne ke liye. (e.g. <td rowspan="2">)
