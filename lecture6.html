Media Tags - Audio, Video, Embed, Iframe



  1. Iframe Tag

    The iframe tag is used to embed another HTML document within the current document.
    It creates a frame inside webpage that can display content from another source, such as a webpage, video, etc.
    Important attributes of the iframe tag include:
    • src: Specifies the URL of the page to embed.
    • width and height: Define the dimensions of the iframe.
    • frameborder: Controls the border around the iframe (0 for no border, 1 for border).
    • allowfullscreen: Allows the iframe content to be displayed in fullscreen mode.

    Example: 
    <iframe src="https://www.youtube.com/embed/NGYp1zCBthk?si=QR9zca6OK01kOFsb" width="50%" height="400" frameborder="0" allowfullscreen></iframe>
    Output is shown as the above Youtube Video.

  2. Audio Tag

    The audio tag is used to embed sound content in a web page. It supports various audio formats like MP3, WAV, and OGG.
    We can control playback with attributes:
    • controls: Displays audio controls (play, pause, volume).
    • autoplay: Starts playing the audio as soon as it is ready.
    • loop: Repeats the audio indefinitely.
    • muted: Mutes the audio by default.

    Example: 
    <audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element. => This runs when browser does not support the audio tag or above audio source formats.
</audio>

  3. Video Tag

    The video tag is used to embed video content in a web page. It supports various video formats like MP4, WebM, and OGG.
    We can control playback with attributes:
    • controls: Displays video controls (play, pause, volume).
    • autoplay: Starts playing the video as soon as it is ready.
    • loop: Repeats the video indefinitely.
    • muted: Mutes the video by default.
    • width and height: Sets the dimensions of the video player.

    Example: 
    <video width="420" controls loop>
  <source src="movie.mp4" type="video/mp4">

Deployment of Multi Page Website

Deployment is the process of making your website live and accessible on the internet. Here are the general steps to deploy a multi-page website: You can use platforms like GitHub Pages, Netlify, Vercel, or traditional web hosting services.
  1. Choose a Hosting Platform: Select a hosting service that suits your needs. For beginners, GitHub Pages or Netlify are great options as they offer free hosting for static websites.
  2. Prepare Your Files: Ensure all your HTML, CSS, JavaScript, images, and other assets are organized in a single directory. Make sure all links between pages are relative paths.
  3. Drag and Drop your project folder in Netlify or Vercel dashboard or push your code to GitHub repository if using GitHub Pages.
  4. Configure Domain (Optional): If you have a custom domain, you can configure it to point to your hosting platform or you can use the free subdomain provided by the hosting service.
  5. Test Your Website: After deployment, visit your website URL to ensure everything is working correctly. Check all links, images, and functionality.
