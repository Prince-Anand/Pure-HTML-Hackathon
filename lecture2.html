HTML : Hyper Text Markup Language

HyperText Markup Language(HTML) is the standard language used to create and structure the content of a webpage.

  • HyperText: It refers to the ability to create links (hyperlinks) that connect different web pages and resources on the internet.

  • Markup Language: HTML is a markup language, which means here we use tags to tell the web browser how to display the content.
    Tags are enclosed in angle brackets, e.g., <tagname>.

HTML Tags

  1. The Heading (<h1> to <h6>)

    2. Headings are tags used to define titles and subtitles on your page.

    • <h1>: The most important heading, typically used only once per page for the main title.
    • <h2>: A major section heading.
    • <h3>: A sub-section heading under an <h2>.
    • ...and so on, down to <h6>, the least important heading.

  2. The Paragraph Tag (<p>)

    3. Used for Grouping sentences and blocks of text together and tell browser that this is a paragraph.

    Browsers automatically add a little bit of space before and after a <p> element, separating it from other content

    Example: <p>This is a paragraph. It contains several sentences about a topic.</p>

  3. Horizontal Rules (<hr>)

    4. The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

    They are self closing tag, means they don't need a </hr> tag.

  4. The Line Break Tag: (<br>)

    5. The <br> tag tells browse to hit the enter key and start a new line within a paragraph or block of text.

    They are also a self closing tag, means they don't need a </br> tag.

    Example: <p>This is the first line.<br>This is the second line.</p>

    Output:

    This is the first line.
    This is the second line.

  5. List Elements (<ul>, <ol>, <li>)

    6. These tags are used to create lists in HTML.

    ul stands for unordered list (bulleted list), ol stands for ordered list (numbered list), and li stands for list item.

  6. The Anchor Tag (<a>)

    7. It is used to create hyperlinks that link one page to another or to a specific section within a page.

    Attributes (extra information given inside a tag):

    • href: Specifies the URL of the page the link goes to.
    • target: Specifies where to open the linked document. Common values are _self (default, opens in the same tab) and _blank (opens in a new tab).

    Example: <a href="https://www.google.com" target="_blank">Go to google</a>

    Output: Go to google

  7. The Image Tag (<img>)

    8. The Image tag is a self-closing tag used to embed an image onto your page.

    Attributes:

    • src: Specifies the path to the image file.
    • alt: Provides alternative text for the image if it cannot be displayed.
    • width and height: Define the dimensions of the image in pixels.

    Example: <img src="image.jpg" alt="Description of image" width="150">
