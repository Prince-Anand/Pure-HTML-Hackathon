HTML Forms

Till now, user can only view the content on our web pages. But what if we want to interact with users and get some input from them? We need ways to collect user input on our websites. This is where HTML forms come into play. Forms allow users to enter data, which can then be sent to a server for processing.

Here, <form> tag comes into play...

The <form> element is used to create an HTML form for user input. It can contain various types of input elements, such as text fields, checkboxes, radio buttons, submit buttons, and more.

Lets go through some important tags and attribute needed to create forms in html:

  • <input> element:

    This helps us to make input fields for user to write something. It is a self closing tag. It has various types like text, password, checkbox, radio, submit etc.

    Example:

    Most common type is text:
    <input type="text" placeholder="Enter your username">

  • name attribute:

    For now when user types something and browser sends the data to server, server doesn't know which data is for which input field. To solve this problem, we use name attribute in input tag. We cant use ID as ID should be unique for each element. But name can be same for multiple elements like checkboxes.

    Example:

    <label for="firstname">First Name:</label>
<input type="text" id="firstname" name="firstname">
    This is shown as:

  • type attribute:

    This attribute specifies the type of input element to display. Some common types are:

    • text: A single-line text input field.
    • password: A text field that masks the input (for passwords).
    • checkbox: A checkbox that can be checked or unchecked.
    • radio: A radio button, used for selecting one option from a group.
    • submit: A button that submits the form data to the server.
    • email: A text field for entering an email address, with basic validation.
    • number: A field for entering numeric values, with up/down arrows for incrementing/decrementing.
    • date: A field for selecting a date from a calendar picker.
    • file: A field for uploading files.

    Example:

    <input type="password" placeholder="Enter your password">

  • <button> element:

    This element represents a clickable button. It can contain text or images, and it can be styled using CSS. The button can be used to submit a form, reset a form, or trigger JavaScript functions when clicked.

    This is better than input type submit as we can add more styling to it.

  • <textarea> element:

    This element is used to create a multi-line text input field. It allows users to enter larger amounts of text, such as comments or messages. The <textarea> element can be resized by the user, and it can also have rows and columns attributes to specify its initial size.

    Example:

    <textarea rows="4" cols="50" placeholder="Enter your message here"></textarea>

  • <select> and <option> elements:

    These elements are used to create a drop-down list of options. The <select> element contains multiple <option> elements, each representing a different choice. Users can select one option from the list.

    Example:

    <select name="courses">
    <option value="System Design">System Design</option>
